Timestamp API

Usage

Send string with date in any format to recieve UNIX timestamp and formatted date (MM DD, YYY) in response body. Can be GET or POST request.

GET https://fierce-sociology.glitch.me/api/timestamp/1500349847

POST https://fierce-sociology.glitch.me/api/timestamp

Url Shortener Microservice

Usage

Create unique short URLs. Persists data to postgresql database in the . Short url parameter generated by converting unique integer to base 36.

GET https://fierce-sociology.glitch.me/api/shorten/https://www.sltls.org

POST https://fierce-sociology.glitch.me/api/shorten
