Timestamp API

Usage

Send string with date in any format to recieve UNIX timestamp and formatted date (MM DD, YYY) in response body. Can be GET or POST request.

GET https://fierce-sociology.glitch.me/api/timestamp/1500349847

POST https://fierce-sociology.glitch.me/api/timestamp

Demo

Request Header Parser Microservice

Usage

GET https://fierce-sociology.glitch.me/api/whoami

Url Shortener Microservice

Usage

Create unique short URLs. Persists data to postgresql database in the . Short url parameter generated by converting unique integer to base 36.

GET https://fierce-sociology.glitch.me/api/shorten/https://www.sltls.org

POST https://fierce-sociology.glitch.me/api/shorten

Demo

PROTIP add a "+" to the end of your short url to get some killer stats

Image Search Microservice

Usage

Append a string to the endpoint to generate a set of 10 images and some juicy metadata. Paginate through the responses by adding ?offset=2 (or any other number) to the URL.

GET https://fierce-sociology.glitch.me/api/shorten/undertale fanart

GET https://fierce-sociology.glitch.me/api/latest/imagesearch/

*Search powered by Microsoft Developer Network.
